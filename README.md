# Solar Powered Iron Press System ☀️👕

** Overview

This project presents a conceptual design of a battery-free solar-powered ironing system developed using thermal concentration and oil circulation mechanisms. The system aims to provide an eco-friendly and cost-effective ironing solution for off-grid and rural applications.

The design uses a Fresnel lens to concentrate solar energy and generate heat, which is transferred through heated oil circulation for ironing operations.

# Objectives

* Develop a sustainable ironing solution using solar energy
* Reduce dependency on electricity
* Improve thermal efficiency using insulation techniques
* Create a low-cost ironing mechanism suitable for rural areas

# Working Principle

1. Solar radiation is concentrated using a Fresnel lens.
2. The concentrated heat increases the temperature of thermal oil.
3. Heated oil circulates through a metallic chamber beneath the iron plate.
4. Heat transfer from the chamber enables ironing without electrical power.
5. Insulation materials minimize thermal losses.
